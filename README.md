# Pet Shop DEV - Landing Page
Uma landing page desenvolvida para praticar TypeScript, Next.js, Shadcn, AOS e Tailwind CSS. A página exibe os serviços de um pet shop fictício com foco em design responsivo, animações e interação com o usuário.

## Tecnologias
- Next.js com TypeScript
- Shadcn para componentes UI
- AOS para animações no scroll
- Tailwind CSS para estilização
- Lucide Icons e Phosphor Icons para ícones
